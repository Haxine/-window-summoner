# 呼来唤去 - 为任意程序、窗口、窗口组设置老板键

<figure>
    <img src="misc/demo1.gif">
    <!-- <figcaption>演示</figcaption> -->
</figure>


## 功能

- 召唤：为任意程序配置老板键，随叫随到
- 绑定：给窗口绑定老板键，来去自如
- 工作区：将当前打开的所有窗口保存为工作区，随时切换

### 可选

- 开机自启、最小化到托盘
- 唤起新窗口时隐藏其他已唤起的窗口
- 用最小化代替隐藏
- 使用正则表达式精准捕捉需要控制的窗口
- 关闭过渡动画
