# 呼来唤去 - 一键显隐窗口

## 功能

- 为任意程序配置老板键，一键唤起 / 隐藏
- 为任意窗口绑定老板键
- 支持开机自启、最小化到托盘
- （可选）唤起新窗口时隐藏其他已唤起的窗口
- （可选）最小化窗口而不是隐藏窗口
- （可选）使用正则表达式精准捕捉需要控制的窗口
- （可选）打开/关闭过渡动画
